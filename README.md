# ✂️ Линейный раскрой материала

Программа оптимального линейного раскроя материала с графическим интерфейсом.

Алгоритм **First Fit Decreasing (FFD)** минимизирует количество заготовок и отходов.

![Python](https://img.shields.io/badge/Python-3.8%2B-blue?logo=python&logoColor=white)
![Tkinter](https://img.shields.io/badge/GUI-Tkinter-green)
![Matplotlib](https://img.shields.io/badge/Визуализация-Matplotlib-orange)

---

## 📦 Возможности

| Функция | Описание |
|---|---|
| **Раскрой FFD** | Алгоритм First Fit Decreasing — оптимальное размещение изделий по заготовкам |
| **Припуски** | Настраиваемые припуски слева и справа от заготовки |
| **Ширина пропила** | Учёт ширины пропила между изделиями |
| **Визуализация** | Цветная карта раскроя с прокруткой |
| **Группировка** | Одинаковые заготовки объединяются (`×8` вместо 8 строк) |
| **Умные подписи** | Если название не помещается — отображается `#номер` |
| **Импорт** | Массовый импорт изделий из текста (многострочный) |
| **Редактирование** | Двойной клик по строке — редактирование изделия |
| **Экспорт PDF** | Полный отчёт с таблицей изделий, результатами и визуализацией |
| **Пустые названия** | Изделия без названия допустимы |

## 📥 Формат импорта

Каждая строка — одно изделие. Два последних числа — длина и количество:

```
Стойка каркасная 1500 4
Перекладина 800 6
Труба 100x50x4 2000 8
1200 3
```

| Ввод | Название | Длина | Кол-во |
|---|---|---|---|
| `Стойка каркасная 1500 4` | Стойка каркасная | 1500 | 4 |
| `Перекладина 800 6` | Перекладина | 800 | 6 |
| `1200 3` | *(пустое)* | 1200 | 3 |
| `1200` | *(пустое)* | 1200 | 1 |

Поддерживается запятая как десятичный разделитель: `10,5` = `10.5`

## 📊 Группировка одинаковых заготовок

Если несколько заготовок раз��ожены одинаково — они отображаются **одной строкой**:

```
Заготовка #1-37 (37 шт.):
    #1 Проверка(1300) ×4шт + #2(50) ×2шт
    → остаток 100 мм, пропилов: 5
```

Вместо 37 одинаковых строк — одна с пометкой количества.

## 📄 Экспорт PDF

Отчёт в формате A4 включает:

1. **Титульная страница** — название материала, параметры, таблица изделий
2. **Результаты раскроя** — текстовый отчёт с группировкой (с разбивкой на страницы)
3. **Визуализация** — цветные карты раскроя (A4 альбом, с группировкой)

---

## ⚠️ Обработка ошибок

- Изделия длиннее заготовки **не блокируют** расчёт — они исключаются, а в конце показывается предупреждение
- В отчёте появляется секция **«НЕ РАЗЛОЖЕНО»**
